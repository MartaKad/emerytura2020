# emerytura2020

Ćwiczenia w ramach szkoły programowania, projekt Emerytura

To jest drugi akapit.

## Podrozdział

Fragment kodu:

```csharp
int x = 1;
Console.Write(x);
```

A teraz kod wtrącony w tekst `int y = 100;`. Ten zapis deklaruje zmienną `y` typu całkowitego i przypisuje jej wartość `100`.
